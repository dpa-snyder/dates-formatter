Roadmap: Road to Web App

5 phases + stretch
1
Logic Extraction
Isolate all date parsing into date_engine.py. No UI imports. Tests pass before and after.
Pending
W-101Audit date-formatter-gui.py — map logic boundary vs UI
W-102Create prod/date_engine.py — pure logic, no side effects, importable standalone
W-103Repoint existing GUI to from date_engine import … — CTk GUI still runs identically
W-104Run ./run-tests.sh — all 13 tests pass, zero regressions before proceeding
2
Flask Backend
HTTP API wrapping date_engine. Routes validate inputs, call engine, return results — no processing in routes.
Pending
W-201Scaffold web/app.py — Flask, 127.0.0.1 only, no LAN exposure
W-202POST /api/load — accept xlsx/csv, return columns + row count + file token (UUID)
W-203POST /api/process — token + column + mode → processed file download, clean temp on response
W-204GET /api/settings — return current settings JSON
W-205POST /api/settings — write updated settings (same keys as dates-formatter-settings.json)
W-206GET /manual — serve user-manual.html
W-207Error handling — all routes return JSON errors, no Flask HTML error pages
3
Frontend
Single-page app in web/templates/index.html. Vanilla JS, no bundler. Clean, native-feeling — not a data tool aesthetic.
Pending
W-301Design system + base template — CSS tokens, dark/light via data-theme, matches existing palette
W-302File upload zone — drag-and-drop + click-to-browse, shows name/row count/column count after load
W-303Column picker — dropdown populated from /api/load response
W-304Mode selector — three modes, per-mode description, output mode toggle (overwrite / copy)
W-305Run + progress + download — spinner → /api/process → browser file download
W-306Settings panel — gear icon, theme + output mode defaults, persist via /api/settings
W-307Error states — inline display, no alert dialogs
W-308User manual link — opens /manual in new tab
4
Launcher
Single entry point: start Flask, find free port, open browser, handle shutdown on Windows and Mac.
Pending
W-401launch.py — auto-pick port, start Flask daemon, poll for ready, open browser
W-402prod/date-formatter-web.bat — Windows launcher
W-403prod/date-formatter-web.command — Mac launcher (chmod +x required)
W-404Shutdown — clean temp dir on KeyboardInterrupt or browser close signal
5
Parity + Polish
Feature parity with CTk GUI, plus improvements only the web stack enables.

W-501Theme persistence — load from /api/settings, save on toggle
W-502Recent files — last 5 filenames in settings, shown under upload zone
